# Actividad: Registro y Procesamiento de Datos de Personas por Línea de Comandos

## Descripción General

Esta actividad consiste en desarrollar un programa que permita ingresar datos de varias personas (nombre, edad y nota) a través de la línea de comandos. El objetivo es practicar la captura de datos, el almacenamiento en estructuras adecuadas y el procesamiento de la información para obtener distintos listados y estadísticas.

## Requerimientos de la Actividad

- **Ingreso de datos:**
  - El usuario debe poder ingresar el nombre, la edad y la nota de cada persona.
  - El proceso de carga finaliza cuando el usuario ingresa la palabra "fin" o "terminado" como nombre.

- **Procesamiento:**
  - Los datos se almacenan en una estructura tipo matriz/lista, donde cada elemento representa una persona con sus tres datos.

- **Salida de información:**
  1. Mostrar el listado completo de personas tal como fueron ingresadas.
  2. Mostrar el listado de personas ordenado por nota, de mayor a menor.
  3. Calcular y mostrar el promedio general de las notas ingresadas.

- **Interfaz:**
  - El sistema debe ejecutarse únicamente por línea de comandos (no web, no escritorio).
  - Los mensajes y las instrucciones deben ser claros e intuitivos para facilitar el uso del sistema.

## Entregables

- Implementación en **JavaScript** (archivo `programa.mjs`).
- (Opcional) Implementación en **Python** si se requiere.

## Ejecución del Programa (JavaScript)

1. Abrir una terminal en la carpeta del proyecto.
2. Ejecutar el siguiente comando:
   ```
   node programa.mjs
   ```
3. Seguir las instrucciones que aparecen en pantalla para ingresar los datos.
4. Al finalizar, se mostrarán los listados y el promedio solicitado.

## Objetivos de Aprendizaje

- Práctica de entrada y salida de datos por consola.
- Uso de estructuras de datos (listas, matrices).
- Ordenamiento y cálculo de estadísticas básicas.
- Desarrollo de programas interactivos y usables desde la terminal.
