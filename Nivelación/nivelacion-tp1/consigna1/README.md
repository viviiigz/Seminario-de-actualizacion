 ## Ejercicio 1: DOM y Eventos con JavaScript Puro

## Pantallazo General del Proyecto
Esta es una aplicación web sencilla desarrollada como práctica para entender la manipulación del DOM (Document Object Model) y la gestión de eventos utilizando únicamente HTML y Vanilla JavaScript, sin depender de servidores web ni librerías externas.


### Funcionalidades Principales
El sitio cuenta con dos interacciones clave:

1. **Caja de Texto Interactiva:** Un `input` que escucha el evento de teclado. Al presionar la tecla `Enter`, el sistema captura el valor ingresado y lo muestra al usuario mediante una ventana emergente (`alert`). Posteriormente, la caja se limpia automáticamente.
2. **Alternador de Temas (Botón de Colores):** Un botón que, al detectar el evento `click`, modifica las propiedades de estilo de la página. Alterna el color de fondo y el color del texto rotando entre tres estados diferentes (Blanco/Negro, Negro/Blanco, Celeste/Rojo).

###  Tecnologías Utilizadas
* **HTML5:** Para la estructura semántica de la página.
* **JavaScript:** Para la lógica de negocio, manejo de estados y escucha de eventos (`keypress`, `click`).

##  Cómo ejecutar el proyecto (Implementación/Consigna 1)

Para probar la aplicación, seguí estos simples pasos:

1. Cloná este repositorio o descargá el archivo `index.html` en tu computadora.
2. Abrí el explorador de archivos y navegá hasta la carpeta donde guardaste el documento.
3. Hacé **doble clic** sobre el archivo `index.html`. 
4. El proyecto se abrirá automáticamente en tu navegador web predeterminado (Chrome, Edge, Firefox, etc.) y estará listo para ser utilizado.
